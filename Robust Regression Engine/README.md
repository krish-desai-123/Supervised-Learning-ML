## 🌟 About the Repository

Welcome to the **Robust Regression Engine**! This project moves beyond standard linear regression to tackle complex, real-world house price prediction using advanced machine learning algorithms.

By analyzing the `Advanced_Regression_HousePrice_Dataset.csv`, this repository compares multiple modeling approaches—including L1/L2 Regularization (Lasso/Ridge), Support Vector Regression (SVR), and ensemble methods (Random Forest). It emphasizes robust evaluation through cross-validation strategies and detailed residual diagnostics to ensure the models are accurate, generalizable, and mathematically sound.

**Topics:** `python` `machine-learning` `advanced-regression` `random-forest` `lasso-ridge` `svr` `data-science` `jupyter-notebook`

## 📊 Key Visualizations & Insights

This project emphasizes model transparency through deep visual diagnostics:

**1. Advanced Model Comparison**
Evaluating performance metrics across all tested algorithms to select the optimal model.

**2. Regularization Analysis (Ridge vs. Lasso)**
Observing how L1 and L2 penalties shrink feature coefficients to prevent overfitting.

**3. Random Forest Feature Importance**
Extracting interpretability from ensemble models to understand the strongest drivers of house prices.

**4. Final Model Diagnostics**
Analyzing residuals and Q-Q plots to ensure the final model meets regression assumptions.

> Additional EDA plots (Correlation Matrices, Cross-Validation Strategy Comparisons) can be found in the `assets/` folder and within the main notebook.

## ⚙️ Getting Started

Want to run these advanced models on your local machine and experiment with hyperparameter tuning? Follow these steps:

### 1. Clone the Repository

```bash
git clone https://github.com/krish-desai-123/Robust-Regression-Engine.git
cd Robust-Regression-Engine
```

### 2. Run the Code

Make sure you have Python installed, then install the required libraries:

```bash
# Install required libraries
pip install pandas numpy matplotlib seaborn scikit-learn jupyter

# Launch the interactive notebook
jupyter notebook
```

Open `robust_regression_engine.ipynb` to explore the code, execute the cross-validation, and train the models!
